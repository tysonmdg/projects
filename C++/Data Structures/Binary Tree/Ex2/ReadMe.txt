This solves a genealogical tree where:
	- Father´s age have to be 18 years bigger than son´s age (if theres any).
	- Second son´s age has to be 2 years lower than the first one.
	- Son´s genealogical tree have to be correct as well.

Example input:
5
40 20 2 -1 -1 0 -1 -1 -1
23 3 -1 -1 2 -1 -1
90 60 35 -1 -1 30 11 -1 -1 8 -1 -1 57 39 15 -1 -1 -1 28 10 -1 -1 6 -1 -1
20 -1 1 -1 -1
-1

Example output:
SI 3
NO
SI 4
NO
SI 0
